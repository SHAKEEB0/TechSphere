# Contributing to TechSphere

Thank you for contributing to TechSphere! This project is designed to be student-friendly and open source.

## How to contribute

- Fork the repository and create a new branch for your feature or bug fix.
- Write clear commit messages.
- Follow PEP 8 for Python code and use consistent formatting.
- Add tests for new behavior.
- Submit a pull request describing your changes and the problem solved.

## Development workflow

1. Create a fork of the repo.
2. Create a branch: `git checkout -b feature/my-feature`.
3. Install dependencies and run the app locally.
4. Run tests: `python manage.py test`.
5. Push your branch and open a pull request.

## Issues

Please open issues for bugs, feature requests, or documentation improvements.
